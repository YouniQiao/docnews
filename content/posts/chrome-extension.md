## 文档伴读（CoRead） - 开发者文档助手

专为开发者文档打造的互动伴侣。支持划词评论、查看同路人的避坑指南、接收互动通知。让阅读文档不再是单机游戏。

下载站点：[文档伴读站点 https://read.harmonyos.cool/](https://read.harmonyos.cool/)

![网站截图](/img/2025-12-24_21-19-49.png)



## 核心用途
“文档伴读”是一款专为开发者量身定制的浏览器扩展。它无缝集成于开发者官网（当前支持鸿蒙开发者文档中心developer.huawei.com），在不破坏原有阅读体验的前提下，在浏览器侧边栏为您开启一个即时的讨论空间。

您是否在阅读官方文档时遇到过晦涩难懂的描述？是否在某个API调用上踩过坑却无处吐槽？或者发现了文档错误想提醒后来者？

“文档伴读”将静态的文档变为动态的社区。它让每一段代码、每一行说明都成为开发者交流的节点。

## 为什么您应该安装它？
🚀 拒绝“单机”开发，共享避坑经验

官方文档往往是标准的、通用的，但实际开发中的“坑”却是千奇百怪的。通过安装此插件，您可以直接在文档对应的位置看到其他开发者留下的备注、警告和解决方案，极大地节省您的排查时间。

✨ 所见即所得的“划词评论”

无需切换页面，无需寻找复杂的反馈入口。只需用鼠标选中您有疑问或想分享的文档段落，悬浮按钮即刻出现。您的评论将精准锚定到该段落，让后续的讨论更加聚焦、高效。

🔔 即时的互动反馈

当您的评论收到回复或点赞时，插件会通过站内信及时通知您。您不再是一个人在战斗，而是在与整个鸿蒙开发者生态互动。

## 主要功能
* 侧边栏伴读模式：不遮挡正文，在屏幕右侧优雅展示与当前文档关联的所有讨论。
*  精准划词引用：支持选中页面文本进行针对性评论，支持高亮显示引用内容。
* GitHub 快捷登录：使用您的 GitHub 账号一键登录，保留您的贡献记录与个人主页。
* 实时互动通知：集成的消息中心，不错过任何一条技术交流与点赞鼓励。
* 跨文档聚合：支持跳转至聚合阅读站点，查看全站所有讨论。

## 隐私与安全
我们深知隐私的重要性。本扩展程序：
* 仅在支持的开发者文档网站激活。
* 仅收集您主动发布的评论内容及必要的公开身份信息（基于 GitHub）。
* 不会读取您的浏览历史或其他网站的敏感数据。
